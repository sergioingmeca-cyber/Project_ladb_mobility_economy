# 🚖 Mobility Economy | Urban Mobility Data Analysis

## 📖 Resumen Ejecutivo

La movilidad urbana desempeña un papel fundamental en el desarrollo económico y la planificación de las ciudades. Analizar datos relacionados con desplazamientos, comportamiento de usuarios y variables económicas permite comprender tendencias que facilitan la toma de decisiones basada en evidencia.

En este proyecto se desarrolló un análisis exploratorio de datos utilizando Python para identificar relaciones entre variables de movilidad y economía urbana, obteniendo información relevante mediante técnicas estadísticas y visualizaciones.

---

# 🎯 Objetivo del Proyecto

Explorar y analizar datos relacionados con movilidad urbana y variables económicas para identificar patrones, tendencias y relaciones que apoyen la toma de decisiones fundamentadas en datos.

# 📂 Descripción del Dataset

El conjunto de datos contiene información relacionada con indicadores de movilidad y variables económicas utilizadas para evaluar diferentes comportamientos dentro del contexto urbano.
